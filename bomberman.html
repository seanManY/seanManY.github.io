Not Bomberman

I know it says “Not Bomberman” but its basicly bomberman, please don't sue me. A network version of bomberman created by a team of 3 for our network programming project. Players can create accounts to and use them to login to a server to play the game over the internet. The client side game was created in unity with the server side using Javascript/ JSON.

We were a smaller team in class so we did not have the time to create fancy power ups and abilities to the game. It’s just bomberman, place a bomb down it explodes try not to die, just like life.
